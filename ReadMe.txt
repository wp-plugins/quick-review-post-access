== Description ==

Adds a link to 'Pending', 'future' (scheduled) and 'Drafts' under the Posts, Pages, and other custom post type sections in the admin side menu. Compatible with WordPress 3.0+.

== Installation ==

1. Upload the .php file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
